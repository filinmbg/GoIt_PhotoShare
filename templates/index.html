Paw Prints - REST API для зберігання та управління фотоальбомів. API побудований з використанням інфраструктури FastAPI та SQLAlchemy для управління базою даних.

Користувачі та фотоальбоми зберігаються в базі даних PostgreSQL. Світлини з альбомів зберігаються з використанням сервісу Cloudinary.
